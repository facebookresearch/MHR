# MHR - Momentum Human Rig

A minimal Python package for the Momentum Human Rig.

## Installation

```bash
pip install mhr
```

## Development

Using [pixi](https://prefix.dev/docs/pixi/overview):

```bash
pixi install
pixi run test
```

## Requirements

- Python >= 3.12
